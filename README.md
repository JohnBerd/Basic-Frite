# BASIC FRITE

Application de réplication des QR codes BASIC FIT

Un QR code Basif FIT peut être scanné par l'app pour récupérer le gros des informations (GUID et carte de membre)
Il faudra récupérer le `deviceId` via `CharlesProxy` ou `ProxyMan` à l'ouverture de l'app Basic FIT d'un utilisateur ayant un abonnement en cours
